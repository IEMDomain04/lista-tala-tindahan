# Tala Tindahan

A TypeScript + DialogFlow application that implements NLP (Natural Language Processing) with a database-driven approach for conversation management.

## Overview

This project combines DialogFlow's conversational AI capabilities with TypeScript to build a intelligent dialogue system, with conversation data and intents stored in a Supabase database for easy management and scalability.

## Tech Stack

- **Frontend**: TypeScript
- **NLP Engine**: DialogFlow (Google Cloud)
- **Database**: Supabase (PostgreSQL)
